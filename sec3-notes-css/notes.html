Practising Positioning

Positioning can go against the flow of a page.

Relative Positioning

Example for relative. You can add top bottom, left or right. gets positioned away from original placement.
position: relative; /* push down from top */ top: 20px; /* push from right to left */ right: 20px;

absolute Positioning

Example for absolute,takes element out of flow of parent element. now can be moved in sync with body element which totally breaks the flow. so you need to give the parent element a relative position so you can move relative to that element.
