HTML Entities

Reserved characters in HTML must be replaced with character entities. Characters that are not present on your keyboard can also be replaced by entities.

Tables

tables have 3 sections head, body and footer. table row tags ( a row of cells that go across the table). In the head you need a row and each cell in the row will be a th tag. In the body you need a row and each cell in the row will be a td (table data) tag. For more content you add more rows. Not really sure about the table footer.
