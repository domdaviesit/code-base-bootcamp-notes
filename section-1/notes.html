Some git notes

command to show hidden files is ls -a

git add . adds anything that is new

git log - shows previous commits

More git commands here

  • Setting up git git config --global user.name "Spiderman"
  • git config --global user.email spiderman@example.com
  • Initializing git git init
  • Adding files git add filename or git add .
  • Commit files git commit -m "Commit message goes in here."
  • Checking the status git status
  • Push changes git push origin master
  • Create a new branch + switch to it git checkout -b branchname
  • Push changes to the branch git push origin branchname

Markdown for git

  • # Big heading
  • ## smaller heading
  • * unordered list item
